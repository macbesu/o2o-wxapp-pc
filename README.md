## Installation

Node.js >= 8.9.4

## Getting Started

Follow the commands listed below.

```bash
$ git clone git@github.com:ivanwarke/o2o-wxapp-pc.git && cd o2o-wxapp-pc
$ npm install

# Run the project at local server (localhost:8080)
$ npm run start

# Build the project
$ npm run build

# Format the code
$ npm run prettier
```
